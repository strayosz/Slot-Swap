# Slot Swap

Веб-приложение для управления сменами кураторов онлайн-школы: планирование смен,
распределение кураторов и обмен сменами через общий пул.

## О проекте

Slot Swap решает реальную задачу координации работы кураторов — назначение на смены,
учёт посещаемости и гибкий обмен сменами между кураторами. Предметная область знакома
по личному опыту работы старшим куратором, поэтому логика проектировалась под настоящие
сценарии, а не абстрактный CRUD.

Проект написан намеренно без Spring — на чистых сервлетах, чтобы вручную реализовать то,
что фреймворк обычно скрывает: маршрутизацию, слоистую архитектуру, работу с пулом
соединений и доступ к данным. Это дало более глубокое понимание того, как устроен
веб-стек под капотом.

## Возможности

- Сессионная аутентификация (логин/пароль) и разграничение прав по ролям
- Смены разных типов: вебинары, ответы в чатах, написание подсказок
- Назначение нескольких кураторов на одну смену
- Учёт посещаемости и комментарии администратора по итогам смены
- **Обмен сменами**: куратор выставляет смену в общий пул, и любой свободный
  куратор может её забрать — отсюда название проекта

## Технологии

- **Java** — серверная логика на сервлетах (Jakarta Servlet API)
- **FreeMarker** — серверный рендеринг страниц
- **JDBC + собственные DAO** — доступ к данным без ORM
- **HikariCP** — пул соединений с БД
- **PostgreSQL** — хранение данных
- **Maven** — сборка
- HTML / CSS / JavaScript — клиентская часть
